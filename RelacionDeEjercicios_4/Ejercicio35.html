Ejemplos de Overflow

overflow: visible;

Este contenido se desbordará del contenedor y será visible fuera de él.

overflow: hidden;

Este contenido se desbordará pero será ocultado y no podrá verse.

overflow: scroll;

Este contenido se desbordará y aparecerán barras de desplazamiento en ambos ejes.

overflow: auto;

Este contenido se desbordará y aparecerán barras de desplazamiento solo si es necesario.

overflow-x: scroll; overflow-y: hidden;

Este contenido tiene desplazamiento horizontal, pero el contenido vertical se ocultará.

overflow-x: hidden; overflow-y: scroll;

Este contenido tiene desplazamiento vertical, pero el contenido horizontal se ocultará.
